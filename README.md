# Convite madrinhas Val & Erick - versão autônoma

Esta versão reduz o risco de falha porque a animação usa um único arquivo `index.html` com CSS, JavaScript e as 4 imagens embutidas dentro do próprio HTML.

## Suba no GitHub

Na raiz do repositório `convite-madrinhas-val-erick`, deixe apenas:

- `index.html`
- `.nojekyll`
- `README.md`
- pasta `assets` contendo somente `og-image.png`

## Estrutura

```
index.html
.nojekyll
README.md
assets/
  og-image.png
```

O link final fica:

`https://erixbr.github.io/convite-madrinhas-val-erick/`
