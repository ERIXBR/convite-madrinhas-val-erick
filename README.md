## Como publicar
1. Apague tudo que está hoje no repositório `convite-madrinhas-val-erick`.
2. Suba **todos** os arquivos deste pacote para a branch `main`.
3. Em `Settings > Pages`, use:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
4. Aguarde alguns minutos.
5. Teste em `https://erixbr.github.io/convite-madrinhas-val-erick/?v=6`

## Observação
Se o WhatsApp não atualizar a prévia na hora, isso normalmente é cache. Teste colando o link com `?v=6`.
