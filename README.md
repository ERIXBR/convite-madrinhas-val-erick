## Como publicar no GitHub Pages

1. Abra o repositório `convite-madrinhas-val-erick`.
2. Apague os arquivos antigos da raiz.
3. Suba **todos** os arquivos desta pasta para a branch `main`.
4. Confirme em `Settings > Pages`:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main`
   - **Folder:** `/ (root)`
5. Aguarde 1 a 5 minutos.
6. Teste o site em:
   - `https://erixbr.github.io/convite-madrinhas-val-erick/?v=5`
